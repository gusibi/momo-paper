# Infographic Template

Surface: `visual_sheet`
Document Shape: `infographic`
Legacy alias: `infographic`

## Fixed section recipe
1. Headline
2. Framing statement
3. Core path
4. Metric cluster
5. Takeaway footer

## Layout notes
- strong reading path
- `dense` density
- compact copy
- numeric emphasis used intentionally, not everywhere

## Example scaffold
```yaml
headline:
  components: [eyebrow, title_block]
core_path:
  pattern: ranking
  components: [step_rail, fact_card]
metric_cluster:
  pattern: metric_cluster
  components: [fact_card, insight_callout]
takeaway_footer:
  components: [footer_meta]
```
