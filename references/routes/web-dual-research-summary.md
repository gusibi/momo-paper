# Research Summary Template

Surface: `web_dual`
Document Shape: `research_summary`

## Fixed section recipe
1. Thesis
2. Summary strip
3. Key findings
4. Evidence blocks
5. Implications
6. Method

## Layout notes
- conclusion first, evidence second
- keep the route shorter and denser than an editorial article
- include method context whenever findings drive action
- maintain browser and PDF readability

## Example scaffold
```yaml
thesis:
  components: [eyebrow, title_block, summary_strip]
key_findings:
  components: [fact_card, body_paragraph]
evidence_blocks:
  components: [chart_frame, insight_callout]
implications:
  components: [body_paragraph, insight_callout]
method:
  components: [method_note, footer_meta]
```
