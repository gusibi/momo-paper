# Slides Explainer Template

Surface: `slides`
Document Shape: `explainer`
Legacy alias: `ppt_slide`

## Fixed section recipe
1. Cover
2. Agenda
3. Problem / context
4. Analysis
5. Data evidence
6. Comparison / tradeoff
7. Recommendation

## Slide rules
- one slide, one point
- short factual title on every slide
- no long paragraphs
- use large type and restrained annotation

## Example slide scaffold
```yaml
slide:
  title:
  point:
  components: [title_block, chart_frame, insight_callout]
  takeaway:
  footnote:
```
