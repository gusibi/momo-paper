Momo Paper 将模板分成两个等级：**正式 Schema 模板**会校验必需章节、字段类型、顺序和引用；**实验示例**使用 free mode，保留开放 block 与既有视觉能力，但暂不承诺结构契约。

三个正式模板同时也是发布门禁：官网构建必须先通过 strict 校验，才会覆盖上一版静态输出。其他示例继续证明同一 renderer 能兼容更广泛的文档结构。

## 正式 Schema 模板

- [Landing Page](/) — 本站首页；hero、能力说明、对比和 CTA 由 `landing-page` 契约约束
- [研究摘要](/demo/research-summary/) — 研究问题、关键发现、影响、方法、限制与来源
- [深度研究报告](/demo/deep-research/) — 执行摘要、证据、反方观点、方法、建议与引用

## Experimental / Free Mode 示例

- [方案 / 执行摘要](/demo/one-pager/) — 四段结构：摘要 → 背景 → 证据 → 建议，含数据卡片
- [白皮书 / 长文](/demo/long-doc/) — 可迭代章节 + 引用块 + 结论，适合深度长文
- [信件 / 推荐信](/demo/letter/) — 信头 → 正文 → 请求 → 签名，正式书信格式
- [作品集](/demo/portfolio/) — 案例卡片 + 指标卡，背景/行动/结果三段式
- [简历 / CV](/demo/resume/) — 经历时间线 + 技能标签 + 项目列表，打印友好
- [幻灯片 / 演示](/demo/slides/) — 16:9 视口，支持封面/对比/结尾多种布局
- [研报 / 估值分析](/demo/equity-report/) — KPI 卡片 + K 线图 + 风险矩阵 + 催化剂时间线
- [更新日志](/demo/changelog/) — 版本列表，按新增/变更/修复/废弃分类，彩色标记
- [流程 / SOP](/demo/process-flow/) — 步骤编号 + 负责人标注 + 分支条件 + 检查清单
- [时间线 / Roadmap](/demo/timeline/) — 竖线时间线，支持完成/进行中/计划三种状态
- [常见问题](/demo/faq-page/) — 分组问答卡片，问题加粗，回答次级文字
- [案例拆解](/demo/case-study/) — 背景 → 问题 → 方案 → 证据 → 结果 → 经验
- [数据报告 / KPI](/demo/stats-report/) — KPI 大数字 + 趋势折线图 + 数据分群
- [信息图](/demo/infographic/) — 大数字展示 + 环形图 + 步骤路径，视觉冲击力强
