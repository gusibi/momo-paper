Momo Paper / Routed Showcase

A routed design system for documents and visual narratives.

Momo Paper 通过 `document type -> route -> template` 的路径来组织文档和可视叙事。下面这些样张不是不同品牌，而是同一套视觉语言下的公开入口、内部 route 与样张资产。

公开类型 15 类
从 one-pager、long doc 到 equity report、changelog、infographic，Momo Paper 现在提供 15 个公开 document type 入口。
已公开样张 5 类
当前样张覆盖 `one_pager`、`infographic`、`process_flow`、`timeline`、`resume`，其余入口通过模板与 route reference 对外开放。
图示原语 14 个
架构图、流程图、象限图、K 线、瀑布图等都已进入 `assets/diagrams/`，作为文档内的 diagram primitives 使用。
01 / one_pager

说明型网页

适合系统介绍、叙事性页面、可滚动报告和结构化着陆页，同时默认保持 print-safe 文档秩序。
02 / infographic

信息图与指标集群

适合排行榜、对比矩阵、核心数字拆解、单页传播型内容。
03 / process_flow

流程图与工作流

适合 agent pipeline、运营流程、交付路径、阶段化执行说明。
04 / timeline

时间线与里程碑

适合 roadmap、项目进展、产品演进、履历历程和事件脉络。
05 / resume

简历与人物档案

适合个人履历、专家卡、演讲者介绍、作者页和顾问档案。
06 / Registry

已收编与候选项

`faq_page`、`case_study`、`research_summary` 已正式进入 document shape；`comparison_matrix`、`topic_cover` 仍保留为 pattern candidate。
