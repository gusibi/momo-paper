信息图适合做“高密度但单结论”的传播页面。

同一套系统里，信息图不需要换一套新审美。它只是把版面密度拉高、把数字做大、把阅读路径压得更明确，同时仍然遵守相同的色彩和字体系。

Best Use
排行榜 / 对比 / 拆解
适合社交传播、单页海报、快速扫读和关键数字表达。
Density Mode
Dense
但“密”不意味着乱，数字和说明必须主次分明。

三组关键数字，用同一种语言说话。

Metric cluster / Single-page emphasis
01 / Output Reach
12 种

同一设计系统可被映射到网页、PPT、报告、信息图、人物卡、流程说明等多种结构。

02 / Drift Control
-68%

当 token 和 route 进入生成链路后，颜色、间距和图表风格的漂移会显著下降。

03 / Reading Speed
+31%

单页信息图通过清晰路径和大数字，可以把关键信息更快送达给第一次阅读者。

例如版本差异、方案取舍、地区差异、年度变化对照。
