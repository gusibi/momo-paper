Foundation

文档、章节与基础排版

这些组件定义一篇 Momo Paper 文档的基本骨架：外层页面、章节、标题组、眉标和正文段落。

mp-document + mp-section

web_dual.explainer

一个最小文档区块

章节之间用稳定的边界和留白表达结构，不依赖装饰性的卡片墙。

Momo Paper 的基础层优先服务长文阅读、打印导出和 agent 生成稳定性。组件名称固定后，样式可以由不同 CSS 模板接管。

mp-section-header

Section Label

章节标题组件

用于每个主要章节的标题、说明和 route 上下文。

mp-title-block

Title Block

结论先行的标题组

适合页面开头、章节开头或幻灯片主标题。

mp-body-paragraph

正文段落保持可读宽度和稳定行高，用于解释背景、证据、方法和结论。它不承担装饰职责，也不把布局规则写入内容节点。

mp-page-header + mp-subtitle

Page Header

页面级主标题

用于站点页面（如设计系统、使用指南、FAQ 等）的顶层标题区。与 mp-title-block 不同，page-header 带底部 border 分隔线，subtitle 字号更大。
