内容强调与文档节奏

这些组件负责摘要、洞察、引用、分隔和页脚元信息，是长文和报告最常见的阅读节奏。

mp-summary-strip

摘要：组件协议把页面结构固定下来，让 CSS 模板可以独立替换视觉风格。

  • 组件名称稳定，便于 validator 检查。
  • 内容和视觉分离，减少 agent 随机发挥。
  • 打印视图仍能保留章节层级。
mp-section-intro

章节引导段落，字号略大于正文，用于概述本章节的核心内容。适合在设计系统文档、长文和报告中作为每章的阅读导航。

mp-insight-callout
Insight

当一个观点应该被快速捕捉时使用 callout；它不是普通卡片，也不应该堆满整页。
