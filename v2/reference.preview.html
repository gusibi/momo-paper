Momo Paper DSL Reference Example

Complete Phase 1 example covering recommended Agent tags.

Momo Paper DSL

Use Markdown for prose and :::tag-name blocks for structured content.

  • Tags are Agent-facing conventions.
  • The runtime parses and renders them generically.
  • Invalid syntax should fail with a useful error.
  1. Write DSL.
  2. Validate DSL.
  3. Render HTML.

Inline formatting supports bold, italic, code, and links.
